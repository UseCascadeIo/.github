## ✦ What is Cascade?

Cascade is a modern infrastructure control panel built for people who want **full control without the mess**.

No fragmented tooling. No bloated UI.
Just a clean, unified interface that adapts to *your* stack and *your* brand.

---

## ✦ Core Features

### Infrastructure

* **VPS & MicroVMs** — Full KVM virtualisation on your own nodes
* **LXC Containers** — Lightweight workloads, same interface
* **Private vNetworks** — Isolated networking between resources

### Platform

* **Managed Databases** — PostgreSQL, MySQL, Redis
* **Reverse Proxy** — Automatic TLS, instant routing
* **DNS Manager** — Full zone control with propagation tracking

### Experience

* **Fully Customisable UI** — Colours, layout, branding
* **Multi-node Control** — Manage everything from one panel
* **API-first Architecture** — Built to integrate, not isolate
